Order KORUZA

Assembled links and DIY KITS can be pre-ordered from our partner company Fabrikor. KIT includes all parts for building one wireless optical system KORUZA link (two units).

To build KORUZA units from DIY KIT you will need to follow step-by-step KORUZA assembling instructions, which are already published and online. KORUZA source documentation is available in our GitHub repositories.
